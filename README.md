# React Functional component Form Handling with Routing

This is a simple **React.js application** that demonstrates:
- React Router DOM navigation
- Form handling using `useState`
- Storing multiple records in **localStorage**
- Displaying data in a table
- Delete functionality
- Bootstrap styling
